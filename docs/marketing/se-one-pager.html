Get Started in 3 Steps

From zero to a finished as-built report in under 10 minutes.

1

Download

Grab the latest release from GitHub:
macOS → .dmg
Windows → .zip

GitHub Releases

2

Launch

macOS: Drag to Applications, right-click → Open on first launch (Gatekeeper).

Windows: Extract ZIP, run vast-reporter.exe (SmartScreen: More info → Run anyway).

Your browser opens automatically at localhost:5173.

3

Connect & Generate

Enter the cluster IP and credentials, run discovery, place switches, and hit Generate. Your report downloads as PDF + JSON.

Optional: enable SSH port mapping and health checks for deeper coverage.
