Get Started in 5 Steps

1
What You Need

Gather your cluster IP (or VMS address), VMS credentials (support user), node SSH password, and switch SSH credentials. Tech port connections use 192.168.2.2.

View Docs
2
Connect to Cluster

Open the Reporter page, select or create a Cluster Profile, and enter your connection settings. Enable "Autofill Default Passwords" for quick setup.

Open Reporter
3
Configure Switches

Choose Auto or Manual switch placement. Run Discovery to find rack switches, then assign positions or add switches manually from the Library.

Configure
4
Run Reporter

Select your operations — Pre-Validation (recommended), Run Reporter (generates the as-built PDF), and optional Health Check. Press Run and monitor progress.

Run
5
Review Results

View or download your PDF and JSON reports. Browse all past results on the Results page, or download complete bundles from One-Shot runs.

View Results
